# Maintainers

This file lists the current maintainers and guidance for contacting or becoming a maintainer for the KKNTrack project.

Current maintainers

  - Contact: maintainer@untidar.ac.id (replace with the real maintainer email)
  - Role: primary code and release maintainer

How to contact
- For general questions or to report non-critical issues, open an issue on the repository.
- For security-sensitive issues, contact the security contact above (replace email) and avoid posting sensitive details in public issues.

Becoming a maintainer
1. Contribute regularly with high-quality PRs that include tests and clear descriptions.
2. Help review other contributors' PRs and respond to issues.
3. After sustained contributions, the team may invite contributors to the maintainers list.

Maintainer responsibilities
- Review and merge pull requests, enforce code quality and tests.
- Manage releases and versioning.
- Triage and respond to issues and security reports.

Editing this file
- To change maintainer info, edit this file and open a pull request. The current maintainers will review and merge.
